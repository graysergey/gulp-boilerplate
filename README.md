##Gulp 4 boilerplate

1. Clone:

    ```
    git clone git@github.com:regulyarniy/gulp-boilerplate.git
    ```

2. Install prerequisites:

    ```
    cd gulp-boilerplate
    npm install
    ```

* Start dev server with hot reload:

    ```npm run start```
    
* Test linters:

    ```npm run test```

* Build and optimize images for production:

    ```npm run build```
